# SFML-Extended
Simple Extensions for the C++ SFML library.

# Included:

## Spritesheets
### Files:
 - Spritesheet.cpp
 - Spritesheet.hpp

### Dependencies
 - None

### Use
The `sfext::Spritesheet` class can be used to break a spritesheet that you have created into `sf::Sprite` elements, which are then stored inside the class.
